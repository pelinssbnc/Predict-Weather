# 🌦️ Predict Weather

## 📌 Proje Hakkında  
Bu proje, zaman serisi (time series) analizi kullanarak hava durumu verileri üzerinden gelecekteki hava koşullarını tahmin etmeyi amaçlamaktadır.

## 🛠️ Teknolojiler  
Python, Pandas, NumPy, Matplotlib, Scikit-learn  

## 🎯 Amaç  
Geçmiş hava durumu verilerine dayanarak gelecekteki hava durumunu tahmin etmek.  

## 📈 Sonuç  
Model, zaman içindeki trend ve değişimleri öğrenerek anlamlı tahminler üretmiştir.
